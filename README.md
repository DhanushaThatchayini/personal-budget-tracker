# Personal Budget Tracker (C)

A command-line based Personal Budget Tracker developed in C to manage daily income and expenses efficiently. The application helps users record financial transactions, track spending, and maintain data using file handling.

## Features

* Add Salary and Extra Income
* Record Fixed and Daily Expenses
* View Detailed Account Statement
* Check Current Balance
* Generate Monthly Summary
* Search Transactions by Date
* Delete Transactions

## Concepts Used

* Structures
* File Handling (Binary Files)
* Pointers
* Modular Programming
* Menu-Driven Programs

## 🛠️ How to Run

Compile the program:
gcc 1_Personal_Budget_Tracker.c -o budget

Run the program:
./budget
